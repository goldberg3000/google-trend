鲍勃·李（Bob Lee，1979年12月20日 - 2023年4月4日）是一位美国商人兼软件工程师，因帮助创建金融服务应用 Cash App 而闻名。他曾担任 Square 的首席技术官和 MobileCoin 的首席产品官。2023年4月4日，李在旧金山林肯山社区被刺杀身亡[1]。2023年4月13日，旧金山警方逮捕了尼玛·莫梅尼（Nima Momeni），指控他杀害了李[2]。 李于1979年12月20日出生于密苏里州圣路易斯[3][4]。他的兄弟是蒂姆·奥利弗·李[5]。李在林德伯格高中上学期间[6]，用 Turbo Pascal 编写了一个 3D 渲染引擎[7]，并因在水球比赛中充满活力而被人们称为“疯狂鲍勃”[8]。李就读于东南密苏里州立大学，并加入了西格玛·奇兄弟会[8]。2001年8月7日，李发布了一个免费程序来保护微软 IIS 服务器免受当时正在迅速传播的“代码红色”蠕虫的攻击[9]。李曾担任东南密苏里州立大学的网页开发人员[10]。到2003年，李受雇于美国电话电报公司担任技术架构师。在 AT&T 工作期间，李开发了面向方面的编程（AOP）框架 dynaop，类似于 Java 的 Spring 框架，并在《甲骨文杂志》上发表了相关内容[11]。 李于2004年10月[12]至2010年1月[13]在 Google 担任软件工程师，并帮助开发了 Android 移动操作系统[14]。2006年，李在 Google 工作期间与凯文·布里利翁（Kevin Bourrillion）共同开发了依赖注入框架 Guice，用于模块化 AdWords[15][16]。李基于自己在 Apache Struts 上的想法开发了 Guice，并同时在 Apache Struts 项目上工作[17]。2008年，李和布里利翁因其在 Guice 上的工作获得了 Jolt 奖[18]。李于2007年调到核心库团队[19]，最终成为该团队的领导者[20]。2009年5月，李与罗德·约翰逊（Rod Johnson）共同创建了一项 Java 依赖注入提案[21]。他还参与了 Java 提案的专家组，该提案增加了 Lambda 表达式，以及另一个为该语言添加并发性的提案[22]。此外，李还参与了 Dalvik 的开发，Dalvik 是 Android 的进程虚拟机[23]。在甲骨文诉 Google 案中，李被传唤为证人[24]。 2010年1月，电子商务公司 Square 招募了李[13]。他成为公司的首席技术官，领导了公司 Android 应用的开发工作，最终接手了 Square iOS 应用的开发工作，接替了现已故的联合创始人特里斯坦·奥蒂尔尼（Tristan O'Tierney）[25]。成为首席技术官后，李从密苏里州圣路易斯的家中搬到了旧金山[26]。2013年，他帮助 Square 建立了 Cash App，当时称为 Square Cash。他于2014年离开了 Square，投资了多家科技初创公司，包括 Clubhouse、SpaceX 和 Figma[20][27]。在 COVID-19 大流行期间，李帮助世界卫生组织开发其应用程序[28]。2021年，他加入了加密货币支付公司 MobileCoin，担任首席产品官[29]。 李在接受手术前患有多指症[7]。2019年，李的母亲去世后，他的父亲搬到了加州米尔谷，与李的家人住在一起。2019年，李和他的妻子克里斯塔（Krista）分居。两人育有两个孩子[30][31]。2022年10月，克里斯塔和李的两个孩子留在湾区，而李则搬到迈阿密，与他的父亲住在一起[32]。 2023年4月4日凌晨，李在旧金山林肯山社区主街 300 街区附近被刺伤胸部

Nima Momeni Found Guilty in Murder of Cash App Founder Bob Lee

据美国全国广播公司湾区分部报道，周二，尼玛·莫梅尼被判犯有二级谋杀罪，他被指控刺死了现金应用创始人鲍勃·李。报道称，旧金山陪审团驳回了这位科技顾问的自卫申辩，他将面临16年到无期徒刑。定于1月10日举行听证会，确定宣判日期。 “我要感谢陪审团在本次审判中所付出的努力，”地区检察官布鲁克·詹金斯在一份声明中表示。“我还想感谢李先生的家人，感谢他们在漫长的寻求正义的道路上对我们办公室的信任和信心。虽然什么也无法挽回李先生的离去，但我希望这个判决能给他的家人和朋友带来一些慰藉，帮助他们从这场毁灭性的损失中恢复过来。李先生深受许多人的爱戴，他的死震惊了我们所有人，并引起了全国范围内的反响。” 莫梅尼的律师托尼·布拉斯表示，判决结果“显然让我们非常失望”，他的团队将评估是否上诉。 2023年4月13日，莫梅尼因李的谋杀被捕，距离李在旧金山街头被发现满身是血并呼救已近两周。现金应用创始人被送往附近的一家医院，后被宣布死亡。 检察官辩称，莫梅尼策划了对李的袭击，将他带到海湾大桥下的一处僻静区域。他们说，莫梅尼对李感到愤怒，因为李向他的妹妹介绍了一名毒贩，她声称该毒贩给了她毒品并性侵了她，莫梅尼刺了李三刀。 据美国全国广播公司报道，莫梅尼作证说，在莫梅尼批评他找脱衣舞俱乐部而不是花时间陪伴家人后，李用刀攻击了他，他为了自卫才动的手。莫梅尼声称，他当时没有意识到自己已经致命地刺伤了李，也没有意识到自己伤了他。 “我们认为今天正义得到了伸张，”李的哥哥蒂姆·奥利弗告诉媒体。“重要的是，我们今天获得了一个有罪判决，尼玛·莫梅尼将被判入狱很长时间。”

Bob Lee verdict: Cash App creator's killer found guilty of second-degree murder | TechCrunch

据美国全国广播公司湾区频道周二报道，旧金山陪审团裁定尼玛·莫梅尼二级谋杀罪名成立，他被指控刺杀了现金应用创始人、前 Block 首席技术官鲍勃·李。陪审团认定莫梅尼一级谋杀罪名不成立，这意味着陪审员认为李的谋杀并非蓄意。李于 2023 年 4 月在旧金山中心区的街道上被刺死。在审判过程中，检察官描绘了李在其遇害前两天与凶手姐姐卡扎尔·莫梅尼吸毒狂欢的戏剧性场景。检察官指控李的朋友兼毒品贩子在狂欢期间性侵了卡扎尔，导致两人发生激烈争吵，最终尼玛用他姐姐家里的厨房刀刺死了李。然而，辩方给出了不同的说法。尼玛·莫梅尼告诉陪审团，在刺杀案发生的晚上，他与李在海湾大桥下散步，试图说服李多花点时间和家人在一起，而不是出去狂欢。辩方声称，李在谈话过程中变得沮丧并用刀攻击了莫梅尼，导致莫梅尼出于自卫反击，从李手中夺走了刀。莫梅尼告诉陪审团，他对李没有任何敌意。然而，事发经过的视频证据表明，莫梅尼才是攻击者，法医检测发现莫梅尼的 DNA 在刀柄上。一级谋杀的刑罚通常比二级谋杀的刑罚更重，但两者通常都伴随漫长的监禁刑期。莫梅尼将面临 16 年到无期徒刑，具体判决将在稍后公布。李遇害时年仅 43 岁，是加密货币公司 MobileCoin 的首席产品官。在此之前，他领导了杰克·多西的 Block 的技术开发，在那里他创建了移动支付服务 Cash App，他还投资了 SpaceX、Clubhouse、Tile 和 Figma。在他 20 多岁时，他帮助 Google 开发了 Android 操作系统。在李遇害时，他的同事和同僚称赞他对科技界的贡献。Figma 首席执行官迪伦·菲尔德指出，李在他 14 岁时就与他见面，并且是 Figma 的早期支持者。多西说，李在创建 Square（后来成为 Block）的过程中发挥了关键作用。李遇害后，许多人最初猜测这位科技高管的死是随机发生的，与旧金山的无家可归和犯罪问题有关。事实上，事实证明李认识凶手。
