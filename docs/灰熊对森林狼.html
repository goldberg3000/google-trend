孟菲斯灰熊在最后一分钟18秒时，依靠贾·莫兰特的关键跳投，以127-125险胜明尼苏达森林狼。贾伦·杰克逊-Jr.在比赛中贡献了33分，带领灰熊取得胜利。莫兰特拿下12分，并在比赛还剩54秒时将比分扳平，最终完成致命一击。德斯蒙德·贝恩贡献了21分，其中包括8投4中三分球。灰熊在进攻篮板球上占据优势，抢下21个篮板球，而森林狼只有11个。这一优势使孟菲斯获得了108次投篮机会，而森林狼只有83次。尽管安东尼·爱德华兹在最后时刻两次错失关键三分球，但森林狼在整场比赛中依然保持着竞争力，双方都没有取得过超过6分的领先优势。唐特·迪文琴佐为森林狼贡献了赛季新高的27分，贾登·麦克丹尼尔斯则贡献了21分。森林狼的3连胜就此终结，他们在比赛还剩不到一分钟，比分战成125平的情况下，由于朱利叶斯·兰德尔出现失误，导致球队丢掉了第19次球权。凯德·康宁汉姆在底特律活塞对阵多伦多猛龙的比赛中，贡献了22分、17次助攻和10个篮板的三双数据，帮助球队以123-114取得胜利。康宁汉姆仅差一次助攻就能追平他在12月16日创造的个人最佳纪录，目前他在活塞队史上的三双次数排名第二。蒂姆·哈达威-Jr.贡献了27分，表现出色，而马利克·比斯利也为活塞队贡献了18分。活塞队最近11场比赛取得了9场胜利。伊曼纽尔·奎克利以25分领跑猛龙队，球队遭遇了5连败，并且最近17场比赛输掉了16场。斯科蒂·巴恩斯同样表现抢眼，贡献了16分和11个篮板的两双数据。德文·布克在菲尼克斯太阳队对阵犹他爵士队的比赛中贡献了34分，带领球队取得胜利，最终比分定格在114-106。布克在比赛上半场表现尤为出色，8投7中贡献了22分，其中包括5投4中的三分球。凯文·杜兰特为布克贡献了25分。尽管在比赛中一度领先16分，但太阳队直到第四节比赛结束前才最终锁定胜局。格雷森·阿伦在第四节比赛中表现出色，贡献了9分，总共得到了14分。自教练迈克·布登霍尔泽调整轮换策略以来，太阳队最近4场比赛取得了3场胜利。劳里·马尔卡宁以24分领跑爵士队，科林·塞克斯顿贡献了20分。尽管由于伤病或休息日，乔丹·克拉克森和约翰·科林斯缺席了比赛，但替补球员斯维·米哈伊柳克和科迪·威廉姆斯站了出来，分别贡献了13分，帮助爵士队在与太阳队实力强劲的阵容对决中，打出了精彩的比赛。

明尼苏达森林狼队（20胜17负）将在周六晚上（美国东部时间晚上8点，联盟通行证）与孟菲斯灰熊队（24胜14负）进行本赛季首次交锋。森林狼队正处在三连胜的势头中，而灰熊队则希望在周四失利后重回胜轨。以下是比赛开始前需要了解的五件事： 1. **明尼苏达队的防守至关重要：**森林狼队过去几个赛季一直将防守作为其优势，今年也不例外。他们以联盟第六的防守效率（每百回合失分109.3分）进入周六的比赛。在最近的连胜中，他们的防守更加出色。在这三场比赛中，森林狼队的防守效率排名联盟第二（每百回合失分102.5分），并且在对手投篮命中率（38.9%）和每场失分（97.3分）方面排名第一。 2. **安东尼·爱德华兹状态火热：**爱德华兹在最近四场比赛中势不可挡，带领森林狼队取得了3胜1负的战绩。在唯一的失利中，安特砍下了职业生涯最高的53分（31投16中，15投10中，12罚11中）。在三场胜利中，他场均得到30分，6.3个篮板，6次助攻和1.7次抢断。他在这段时间的投篮命中率很高，三分球命中率高达54%（50投27中），本赛季三分球命中率为42.7%。他场均25.7分排名联盟第八。 3. **唐特·迪文琴佐找回状态：**迪文琴佐在最近三场比赛中取代了迈克·康利进入森林狼的首发阵容，帮助球队取得了3连胜。这位第七年的后卫作为首发球员场均得到12.3分，4.3个篮板，4次助攻和1.7次抢断，三分球命中率超过39%（场均7.7次出手），所有数据都是职业生涯新高。 4. **孟菲斯的新秀们正在发挥作用：**灰熊队从他们的首发球员杰伦·韦尔斯和扎克·埃迪那里获得了大量贡献，他们都进入了我们2024-25赛季凯亚新秀排行榜的前五名。韦尔斯本赛季参加了灰熊队的每一场比赛，在所有新秀中得分排名第三（场均11.6分），三分球命中率超过37%。埃迪缺席了一些比赛，但他得分排名第六（场均10.3分），篮板排名第二（场均7.5个），并且作为新秀以四次两双并列第二。 5. **灰熊队的替补席十分强大：**孟菲斯队从他们的替补席获得了很大的提升。他们以联盟第一的替补得分（场均47.1分），助攻（场均13.6次），上场时间（场均21.6分钟）和三分球命中率（40.9%）排名联盟第一，帮助他们获得了联盟第三的净胜分（3.6分）。桑蒂·阿尔达马领衔替补球员，场均得到13分，7.3个篮板和2.9次助攻，三分球命中率接近39%，所有数据都是职业生涯新高。
