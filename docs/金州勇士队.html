金州勇士队

更新时间: 2025-01-09 14:15

Golden State Warriors - Wikipedia

金州勇士队是一支位于旧金山的美国职业篮球队，隶属于美国国家篮球协会 (NBA) 西部联盟太平洋赛区。球队成立于1946年，最初名为费城勇士队，并在1962年迁至旧金山湾区，并以城市命名为旧金山勇士队。1971年，球队更名为金州勇士队。[a][b] 球队的主场位于大通中心。勇士队赢得了1947年的第一届篮球协会 (BAA) 冠军，[c]并在1956年再次夺冠，当时由名人堂三人组保罗·阿利辛、汤姆·戈拉和尼尔·约翰斯顿领衔。1965年1月，球队将明星球员威尔特·张伯伦交易出去后，在1964-65赛季取得了联盟最差战绩（17胜63负）。然而，他们的重建期很短，因为他们在交易张伯伦四个月后就选中了里克·巴里。巴里和贾马尔·威尔克斯带领勇士队在1975年取得了第三座总冠军，这被广泛认为是NBA历史上最大的冷门之一。球队在1980年代表现不佳，但在90年代初随着“跑轰三巨头”蒂姆·哈达威、米奇·里奇蒙德和克里斯·穆林的崛起，重新成为季后赛常客。在斯蒂芬·库里、克莱·汤普森和德雷蒙德·格林的带领下，球队在2015年重返总冠军宝座。2016年，勇士队以73胜9负的战绩创下了NBA常规赛最佳战绩，但在总决赛中以4比3的比分输给了克利夫兰骑士队。在签下前MVP凯文·杜兰特后，球队在2017年和2018年连续两年夺冠。他们在2019年总决赛中输给了多伦多猛龙队，杜兰特在休赛期离队。在接下来的两个赛季中无缘季后赛后，勇士队在2022年重返季后赛，并在总决赛中战胜了波士顿凯尔特人队。勇士队王朝在过去的八年中赢得了四座总冠军，并从2015年到2019年连续五年进入总决赛（并在八年内六次进入总决赛）。勇士队的昵称是“杜布斯”，是“W's”（胜利）的简称。[10][11][12] 勇士队保持着多项NBA纪录：常规赛最佳战绩、单赛季总胜场数（常规赛和季后赛总计）以及季后赛最佳连胜。库里和汤普森通常被认为是史上最伟大的后场搭档。[13][14] 勇士队是NBA总冠军次数第三多的球队，也是总决赛次数第三多的球队，仅次于洛杉矶湖人队和波士顿凯尔特人队。威尔特·张伯伦和斯蒂芬·库里曾在勇士队效力期间获得过NBA最有价值球员奖，共计三次MVP。勇士队成立于1946年，名为费城勇士队，是篮球协会 (BAA) 的创始成员之一。球队的老板是彼得·A·泰勒，他也是美国曲棍球联盟的费城火箭队的老板。[15] 泰勒聘请了在费城地区有着长期篮球推广经验的埃迪·戈特利布担任主教练兼总经理。[16] 球队以1925年在美式篮球联盟效力的费城勇士队命名。[17] 在早期的得分王乔·福克斯的带领下，球队在1946-47赛季的第一届联赛中战胜了芝加哥雄鹿队，以4比1的比分获得了冠军。NBA是由1949年的一次合并而诞生的，正式承认这作为他们自己的第一个冠军。[c] 戈特利布在1951年买下了这支球队。勇士队在1955-56赛季在费城再次夺冠，以4比1的比分战胜了堡垒韦恩活塞队。这个时代勇士队的明星球员是未来的名人堂成员保罗·阿利辛、汤姆·戈拉和尼尔·约翰斯顿。1959年，球队签下了选秀球员威尔特·张伯伦。他被称为“长人张伯伦”，六次成为球队得分王，
