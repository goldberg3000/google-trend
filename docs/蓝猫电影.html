Bluey animated film announced by Disney+ and BBC

凭借其在英国电影和电视艺术学院奖（BAFTA）获奖的儿童电视节目中扮演主角，小狗布鲁伊将首次登上大银幕，出演其首部电影长片。这部电影将于2027年在影院上映，随后将在澳大利亚的Disney+、ABC iView和ABC Kids平台上流媒体播放。BBC工作室（BBC的商业部门）与迪士尼合作制作了这部影片。布鲁伊是一只蓝色的澳洲牧牛犬，和她的妈妈、爸爸和妹妹宾果一起生活。她将生活中的日常琐事变成一个个奇妙的冒险。该节目的创作者乔·布鲁姆表示：“我一直认为布鲁伊值得拥有自己的电影。我希望这部电影能成为一个让全家共同享受的体验。”这部电视剧由澳大利亚制作公司Ludo（蓝色的澳洲牧牛犬是一种澳大利亚犬种）于2017年共同委托ABC和BBC工作室制作。该剧在140多个国家和地区播出，是英国CBeebies和Disney+平台上的头号节目。据尼尔森媒体研究公司的数据，它也是今年全球在Disney+上观看次数最多的电视剧。该剧还获得了众多奖项，包括2023年和2024年的电视评论家协会奖以及今年的皮博迪奖。在过去的一年里，该剧在美国Disney+上的观看时长超过了200亿分钟，使其成为该国按观看时长计算的排名前十位的流媒体节目。这部电影将沿用电视剧中的配音演员，包括梅拉妮·扎内蒂和大卫·麦克科马克分别为布鲁伊的父母奇利和班迪特·希勒配音，电影配乐将由作曲家乔夫·布什创作。

'Bluey' Movie Set for 2027 Theatrical Release From Disney, BBC Studios

由埃利斯·谢弗创作的动画剧集《蓝猫》将登上大银幕。BBC Studios 和华特迪士尼工作室于周二在“早安美国”节目中宣布，这部改编自澳大利亚热门儿童动画剧集的电影将于 2027 年在全球院线上映。电影在影院上映后，将登陆迪士尼+平台，以及澳大利亚的 ABC iview 和 ABC Kids 平台进行流媒体播放。 《蓝猫》电影由剧集创作者乔·布鲁姆编剧并执导，由 Ludo Studio 制作。电影将邀请剧集配音演员参与，包括梅拉妮·扎内蒂和戴维·麦克科马克。剧集作曲家乔夫·布什将为电影配乐。理查德·杰弗里（《蓝猫》第一季至第三季）将与安伯·奈史密斯（《快乐的大脚》，《乐高大电影》）共同执导，奈史密斯担任制片人。布鲁姆将与 Ludo Studio 的查理·阿斯平沃尔和戴利·皮尔森以及 BBC Studios 的贾斯汀·弗林共同担任执行制片人。 根据新闻稿，电影将“继续讲述蓝猫的冒险故事，蓝猫是一只可爱、精力充沛的蓝色牧羊犬，与她的妈妈、爸爸和小妹妹宾果生活在一起。”自 2018 年首映以来，这部剧集已成为全球现象，获得了艾美奖和英国电影和电视艺术学院奖，并成为 2024 年全球迪士尼+ 上观看次数最多的剧集。根据尼尔森数据，它也是当年美国观众总观看次数排名第一的流媒体剧集。 布鲁姆在一份声明中表示：“我非常享受在第三季的《标志》中使用更长的格式进行创作的体验，所以制作一部剧情片感觉是这种体验的自然延伸。我一直认为《蓝猫》应该拍成一部院线电影。我希望这部电影能够成为一个体验式的事件，让整个家庭都能一起享受。我很高兴能继续与塞西莉亚·佩尔森、汤姆·法塞尔和 BBC Studios 团队，以及达娜·沃尔登、艾伦·伯格曼和迪士尼合作，将这部新的《蓝猫》故事搬上大银幕。” 阿斯平沃尔和皮尔森补充道：“在过去的八年里，我们有幸与 50 多位才华横溢的艺术家和制作人组成的团队合作，他们的才华、奉献精神和创造力共同打造了今天备受喜爱的《蓝猫》。他们的工作为将《蓝猫》的世界扩展到电影银幕上创造了绝佳的机会。我们很高兴继续展示昆士兰州作为世界级人才的摇篮，并与全世界家庭分享由乔·布鲁姆创作的全新、精彩的《蓝猫》故事。” 迪士尼娱乐联合主席艾伦·伯格曼和达娜·沃尔登表示，《蓝猫》以其关于游戏、想象力和连接的真挚故事，团结了“世界各地的家庭。我们非常自豪能扩展与 BBC Studios 的合作关系，将《蓝猫》的魔力搬上大银幕——在现实生活中！——我们对乔·布鲁姆怀有深深的感激，因为他创造了一个不断激励和愉悦世界各地的家庭的世界。”
