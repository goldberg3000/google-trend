'Bluey' Movie Set for 2027 Theatrical Release From Disney, BBC Studios

由埃利斯·谢弗创作的动画剧集《蓝猫》将登上大银幕。BBC Studios 和华特迪士尼工作室宣布，这部取材于澳大利亚热门儿童动画剧集的电影将于 2027 年在全球范围内上映，消息于周二在“早安美国”节目中发布。电影上映后，将在 Disney+ 和澳大利亚的 ABC iview 和 ABC Kids 平台上上线。 《蓝猫》电影由剧集创作者乔·布鲁姆编剧并执导，由 Ludo Studio 制作。电影将沿用剧集原班配音演员阵容，包括梅拉妮·扎内蒂和戴维·麦科马克。剧集作曲家乔夫·布什将为电影配乐。理查德·杰弗里（《蓝猫》第一季至第三季）将与安伯·奈史密斯（《快乐的大脚》、《乐高大电影》）共同担任导演。布鲁姆将与 Ludo Studio 的查理·阿斯宾沃尔和戴利·皮尔森以及 BBC Studios 的贾斯汀·弗林共同担任执行制片人。 据新闻稿称，电影将“继续讲述蓝猫的冒险故事，蓝猫是一只可爱、精力充沛的蓝色牧羊犬，与妈妈、爸爸和妹妹宾果生活在一起”。自 2018 年首播以来，《蓝猫》已成为全球现象级作品，获得了艾美奖和英国电影电视艺术学院奖，并在 2024 年成为全球范围内 Disney+ 上观看人数最多的剧集。根据尼尔森数据显示，它也是当年美国所有观众中排名第一的流媒体剧集。 “我很享受在第三季的《标志》中尝试更长的形式，所以拍摄一部电影感觉像是自然而然的延伸。”布鲁姆在一份声明中说道。“我一直认为《蓝猫》值得拍一部电影。我希望这部电影成为全家人一起享受的体验。我很高兴继续与塞西莉亚·佩尔森、汤姆·弗塞尔以及 BBC Studios 团队，还有达娜·沃尔登、艾伦·伯格曼和迪士尼合作，将这个新的《蓝猫》故事搬上大银幕。” 阿斯宾沃尔和皮尔森补充道：“过去八年，我们有幸与超过 50 位艺术家和制作人组成的非凡团队合作，他们的才华、奉献精神和创造力将《蓝猫》打造成如今备受喜爱的剧集。他们的工作为将《蓝猫》的世界扩展到银幕上创造了条件。我们很高兴继续将昆士兰州打造为世界级人才的中心，并与全球各地的家庭分享乔·布鲁姆创作的全新美丽的《蓝猫》故事。” 迪士尼娱乐联合主席艾伦·伯格曼和达娜·沃尔登表示，《蓝猫》“凭借其关于玩耍、想象力和连接的真挚故事，团结了世界各地的家庭。我们非常自豪地与 BBC Studios 扩大合作关系，将《蓝猫》的魅力带到大银幕上——让它成为现实！——我们对乔·布鲁姆充满感激，是他创造了这个不断激励和取悦世界各地的家庭的世界。”
