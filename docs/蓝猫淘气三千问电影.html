Huge News: A Bluey Movie is Coming! - Bluey Official Website

BBC Studios 和华特迪士尼公司今日宣布，将推出首部《蓝猫》动画电影。这部电影将于2027年在全球影院上映，随后将在 Disney+ 上线。电影还将在澳大利亚影院上映后在 ABC iview 和 ABC Kids 上播出。 这部电影由《蓝猫》的创作者 Joe Brumm 编剧并执导，由 Ludo Studio 与 BBC Studios 合作制作。电影将由剧集中的配音演员参演，包括 Melanie Zanetti 和 David McCormack，以及由《蓝猫》作曲家 Joff Bush 创作的音乐。这部 CG 动画电影将由曾获艾美奖的动画制作人 Amber Naismith（《快乐的大脚》、《乐高大电影》、《乐高蝙蝠侠大电影》）担任制作人，Richard Jeffery 担任联合导演（《蓝猫》第一季至第三季）。Joe Brumm 将与 Ludo Studio 的 Charlie Aspinwall 和 Daley Pearson 以及 BBC Studios 的 Justine Flynn 共同担任执行制片人。 《蓝猫》的创作者 Joe Brumm 表示：“我在第三季的《信号》中体验了更长的格式，因此制作一部电影感觉是自然的延伸。我一直认为《蓝猫》应该拍一部电影。我希望这部电影能成为一个让全家人一起享受的体验式活动。我很高兴能继续与 Cecilia Persson、Tom Fussell 和 BBC Studios 的团队以及 Dana Walden、Alan Bergman 和迪士尼合作，将这个全新的《蓝猫》故事搬上大银幕。” Ludo Studio 联合创始人兼导演 Daley Pearson 和 Charlie Aspinwall 说：“在过去的八年里，我们有幸与超过 50 位才华横溢的艺术家和制作人组成的团队合作，他们的才华、奉献精神和创造力共同塑造了今天的《蓝猫》系列。他们的工作为我们提供了这个绝佳的机会，将《蓝猫》的世界扩展到银幕上。我们很高兴继续展示昆士兰州作为世界级人才的摇篮，并将 Joe Brumm 创作的全新、精彩的《蓝猫》故事与全球家庭分享。”

'Bluey' Movie Set for 2027 Theatrical Release From Disney, BBC Studios

由埃利斯·谢弗执导的热门澳大利亚动画儿童系列剧《蓝猫》将登上大银幕。据《早安美国》周二透露，一部基于该系列剧的电影将于 2027 年由 BBC Studios 和华特迪士尼影业在全球院线上映。影片在电影院上映后，将在 Disney+、澳大利亚 ABC iview 和 ABC Kids 上线播放。《蓝猫》电影由该系列剧的创作者乔·布鲁姆编剧执导，由 Ludo Studio 制作。它将邀请该系列剧的配音演员阵容，包括梅兰妮·扎内蒂和大卫·麦科马克。该系列剧的作曲家乔夫·布什将为影片配乐。理查德·杰弗里（《蓝猫》第一季至第三季）将与安伯·奈史密斯（《快乐的大脚》、《乐高大电影》）共同执导。布鲁姆将与 Ludo Studio 的查理·阿斯平沃尔和戴利·皮尔森以及 BBC Studios 的贾斯汀·弗林共同担任执行制片人。 据新闻稿透露，该电影将“继续讲述蓝猫的冒险故事，蓝猫是一只可爱、精力充沛的蓝边牧羊犬，和她的妈妈、爸爸和小妹妹宾果一起生活”。自 2018 年首播以来，该系列剧已成为全球现象，获得了艾美奖和英国电影学院奖，并成为 2024 年全球 Disney+ 上观看次数最多的剧集。根据尼尔森数据，该系列剧也是 2024 年全美观看人数最多的流媒体剧集（数据截至 11 月）。 “我真的很享受在第三季的‘The Sign’中使用更长的格式进行创作，所以一部电影的制作感觉是对这一形式的自然延伸，”布鲁姆在一份声明中说。“我一直认为‘蓝猫’应该拍一部电影。我希望这是一场全家人都可以享受的体验式活动。我很高兴继续与塞西莉亚·珀森、汤姆·弗塞尔和 BBC Studios 的团队，以及达纳·沃尔登、艾伦·伯格曼和迪士尼合作，将这部新的‘蓝猫’故事搬上大银幕。” 阿斯平沃尔和皮尔森补充说：“在过去的八年中，我们有幸与超过 50 位才华横溢的艺术家和制作人组成的杰出团队合作，他们的才华、奉献精神和创造力共同塑造了今天的‘蓝猫’。他们的工作为‘蓝猫’登上大银幕铺平了道路。我们很高兴继续将昆士兰州展示为世界级人才的摇篮，并与世界各地的家庭分享乔·布鲁姆创作的新的美丽‘蓝猫’故事。” 迪士尼娱乐联席董事长艾伦·伯格曼和达纳·沃尔登表示，“蓝猫”已凭借其关于玩耍、想象力和连接的真情故事将“世界各地的家庭团结在一起。我们非常自豪能够扩大与 BBC Studios 的合作关系，将‘蓝猫’的魔力搬上大银幕——在现实生活中！——我们非常感谢乔·布鲁姆创造了一个继续激励和 delight 世界各地家庭的世界。”

Bluey Movie Coming From Disney, BBC Studios in 2027

热门家庭动画系列《蓝猫》将推出大电影，将于 2027 年上映。 这部由迪士尼公司和英国广播公司工作室联合制作的 CGI 动画电影，将由《蓝猫》的创作者乔·布鲁姆执导并编剧。电影将“继续讲述蓝猫的故事，她是一只可爱、精力充沛的蓝色牧羊犬，与她的妈妈、爸爸和小妹妹宾果生活在一起”。 这部电影将于 2027 年在迪士尼旗下上映，并在上映后登陆迪士尼+流媒体平台。电影将邀请原版动画系列中的配音演员参演，包括梅拉妮·扎内蒂和戴维·麦科马克，分别为蓝猫的父母奇利和班迪特·希勒配音。音乐将由《蓝猫》的作曲家乔夫·布什创作。 《蓝猫》已成为迪士尼+的热门节目，经常位列该平台上播放量最高的节目之一。一个名为“The Sign”的特别篇时长更长，在全球获得了超过 1000 万次的观看量，之后还发布了一些新剧集。 《蓝猫》也为英国广播公司工作室带来了可观的收益，该工作室是英国广播公司的盈利部门，收益来自商品销售和授权。 这部由布鲁姆在他的家乡澳大利亚创作的动画系列，迅速在全球范围内受到儿童和家庭的喜爱。观众们被剧中对想象力和创造力的赞颂以及强大的情感内核所吸引。 迪士尼娱乐联合主席达娜·沃尔登和艾伦·伯格曼在一份声明中表示：“自《蓝猫》在迪士尼少儿频道和迪士尼+ 首播以来，它已成为一种全球现象，以其真挚的玩乐、想象力和连接的故事，将世界各地的家庭团结在一起。我们非常自豪能够与 BBC Studios 扩大合作关系，将《蓝猫》的魔力带到银幕上——让它成为现实！我们也非常感谢乔·布鲁姆，是他创造了这个持续激励和愉悦世界各地家庭的世界。” 布鲁姆补充道：“我在第三季制作《The Sign》时，非常享受创作更长篇幅作品的体验，因此拍摄一部故事片感觉是对这种创作形式的自然延伸。我一直觉得《蓝猫》应该有一部电影。我希望这部电影能够成为一场全家人都能一起享受的体验式活动。我很高兴能够继续与塞西莉亚·佩尔森、汤姆·富塞尔以及 BBC Studios 的团队，以及达娜·沃尔登、艾伦·伯格曼和迪士尼合作，将这个全新的《蓝猫》故事搬上银幕。” 迪士尼和 BBC Studios 将与 Ludo Studio 联合制作这部电影，由经验丰富的动画制作人安伯·奈史密斯（《乐高大电影》、《快乐的大脚》）担任制作人。迪士尼的真人电影和二十世纪影业负责人大卫·格林鲍姆将负责监督迪士尼的制作工作。
