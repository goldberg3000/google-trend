Montana State Bobcats football - Wikipedia

蒙大拿州立大学野猫足球队代表蒙大拿州立大学参加NCAA一级足球锦标赛分区中的大天空会议。该项目始于1897年，共获得三次全国冠军（1956年、1976年和1984年）。它是美国唯一一支在三个不同级别的比赛中获得全国冠军的大学足球队，分别是NAIA、NCAA二级和NCAA一级AA（现为FCS）。截至2022赛季，野猫队共参加了1049场比赛，总战绩为525胜492负32平。[3] 第一次夺冠发生在蒙大拿州立大学在落基山体育联盟的最后一个赛季，该联盟于1952年加入NAIA。这次全国冠军是RMAC的第一个全国冠军，也是NAIA首次举办足球冠军赛。野猫队从1917年到1956年都是RMAC的成员，在此之前，他们在1897年到1916年期间是独立球队。蒙大拿州立大学在1957年重返NCAA（大学组），并在1957年到1962年作为独立球队取得了巨大成功，连续六个赛季取得了胜利，其中包括1957年的8胜2负和1958年的8胜1负。1963年，蒙大拿州立大学成为大天空会议的创始成员，并自此获得了两次全国冠军。[4] 蒙大拿州立大学共获得了21次联盟冠军，包括16次大天空会议冠军和5次落基山体育联盟冠军。野猫队在过去九十年中的八年中获得了联盟冠军，在过去八十年中的七年中获得了多次联盟冠军。蒙大拿州立大学在1926赛季的RMAC联盟比赛中保持不败，但没有获得联盟冠军。他们十三次晋级NCAA季后赛，其中一次（1976年）是作为二级成员，十二次（1984年、2002年、2003年、2010年、2011年、2012年、2014年、2018年、2019年、2021年、2022年和2023年）是作为一级AA/FCS成员。截至2023赛季，野猫队在季后赛中取得了18胜13负2平的战绩。[5] 他们的主要对手是蒙大拿灰熊队，他们每年都会在“荒野斗殴”中相遇，蒙大拿州立大学的学生、校友和支持者更常称之为“猫熊大战”。[6] 在二战之前，蒙大拿州立大学足球队经常取得成功，但只是一些小胜利。从1931年到1941年，蒙大拿州立大学的最后一支战前球队没有取得过一个胜季。在这段时期，“猫队”在对蒙大拿的比赛中取得了1胜10负，并且连续八年被零封。然而，在1946年，情况开始发生变化。由经历过战争的退伍军人克莱德·卡彭特率领的野猫队在常规赛中取得了5胜3负1平的战绩，这一成绩足以让球队获得第一个碗赛资格。野猫队以13比13战平了新墨西哥州立大学。虽然蒙大拿州立大学要再等七个赛季才能再次取得胜多负少的战绩，但那个赛季帮助球队走上了正轨，并开启了蒙大拿州立大学足球历史上一个非常成功的时期，因为野猫队在接下来的30场比赛中，从1956年到1985年，赢得了对灰熊队的22场比赛，并在同一时期赢得了全部三次全国冠军。[需要引证] 然而，1946年的球队之所以特别，不仅在于他们的成就。在二战结束后，球队重新组建，其特殊的使命是在蒙大拿州立大学的前一支球队（1941年球队）的14名成员在二战期间阵亡后，继续高举野猫队的旗帜。在海港碗赛中，唯一一名战前常任队员是比尔·祖潘，他的兄弟阿尔就是那些为野猫队做出最终牺牲的球员之一。其他成员包括奥林·贝勒、纽厄尔·伯格[7][8]、达纳·布拉德福德、约翰·伯克、伯纳德·克鲁赞、威廉·科伊
