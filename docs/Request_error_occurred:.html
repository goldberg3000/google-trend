前特别检察官杰克·史密斯在其关于特朗普试图推翻2020年总统大选的最终报告中得出一个简单而强有力的结论：有足够的证据指控唐纳德·特朗普。这份长达137页的报告在特朗普宣誓就任第二任总统的前一周发布，是对史密斯调查工作的全面辩护，也是针对其众多批评者的反驳。尽管史密斯面临着巨大的政治和法律阻力，尤其是来自最高法院的压力，但他表示，他最终的失败是在选举中特朗普赢得连任之时。根据司法部的政策，史密斯写道，由于特朗普即将获得作为现任总统的豁免权，因此一切都已经结束了。第二卷总结了史密斯另一项工作的成果，即关于特朗普在卸任后保留机密文件和妨碍司法公正的强有力但现已结案的案件，目前尚未公布。就像特朗普在过去几年中赢得的所有法律诉讼一样，这也是2024年大选中的一个终极“假设”问题。以下是史密斯报告中的要点：这是第二次有特别检察官调查特朗普，并指出司法部的政策意味着这位前总统和未来的总统将不会面对陪审团。“司法部认为，宪法禁止对总统继续提起指控和起诉，这种观点是绝对的，并不取决于所指控罪行的严重程度、政府证据的强度或起诉的优劣，而办公室完全支持这一观点。”史密斯写道。相关文章 失落的岁月：梅里克·加兰领导的司法部如何错过了对特朗普进行1月6日事件起诉的机会 “事实上，要不是特朗普当选并即将重返总统职位，办公室认为，可采纳的证据足以在审判中获得和维持定罪。”司法部长梅里克·加兰因直到2022年11月才任命特别检察官调查特朗普而遭到左派的批评——史密斯直到下一年年初才开始工作。这并非一个简单的法律案件。然而，特别检察官表示，2024年大选意味着他已无时间和选择。然而，史密斯并没有保证定罪——他只是说有足够的证据。 “任何检察官在提起任何指控之前都必须得出同样的结论，即有足够的证据让陪审团做出有罪判决，”CNN资深法律分析师埃利·霍尼格说。“他并没有夸口预测假设的陪审团会做什么。”特朗普自2021年1月6日以来一直坚持自己没有做错任何事，现在计划赦免那天在国会大厦暴乱中被逮捕、指控和定罪的许多人，他一再表示自己已经被免罪，因为指控被撤销。特朗普的律师甚至在写给司法部长的信中加入了“完全免罪”这一行，试图阻止报告的发布。但史密斯表示，情况绝非如此。“特朗普先生的信声称，他的刑事案件被驳回表明特朗普先生‘完全免罪’，”史密斯写道。“这是错误的。”特朗普和公众以前见过这样的说法。前特别检察官罗伯特·穆勒在其关于特朗普竞选团队与俄罗斯在2016年大选中是否存在勾结的报告中表示，有足够的证据指控特朗普犯有妨碍司法公正罪，但他没有这样做。“虽然这份报告没有得出总统犯了罪的结论，但也没有对他进行免罪，”穆勒在2019年特朗普第一任期期间写道。“如果我们在对事实进行彻底调查后确信总统明确没有犯下妨碍司法公正罪，我们将明确说明。”穆勒表示。尽管特朗普对联邦指控表示不认罪，并一直称调查是“政治迫害”，但他的律师并没有对史密斯报告的事实提出任何反驳，特别检察官表示。史密斯写道，特朗普的律师写给加兰的信“没有指明对他们审查过的报告草案有任何具体的事实上的异议”。虽然史密斯声称他可以获得定罪，但他承认一个显而易见的事实：最高法院是决定性因素。“由于前所未有的事实和将在本案中进行诉讼的各种法律问题，办公室意识到，本案将涉及诉讼风险，就像任何规模和复杂程度的案件一样，”史密斯写道。“然而，在对法律进行详尽而详细的审查后，办公室得出结论，这些指控得到了充分的支持，并且将在没有法律发生变化的情况下经受住任何法律挑战，前提是法律在起诉时存在的法律。”他补充道。相关文章 特别检察官报告谴责特朗普在2020年“为留住权力而进行的犯罪行为” 但法律
