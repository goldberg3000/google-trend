圣何塞州立大学斯巴达人足球队代表圣何塞州立大学参加美国大学体育协会（NCAA）一级美式足球（FBS）比赛，是山区西部联盟（Mountain West Conference）的成员。自1898年首次参加常规赛以来，该队培养了90多名全美明星球员，赢得18次联盟冠军，并送出139名球员进入国家橄榄球联盟（NFL），其中包括职业橄榄球名人堂成员比尔·沃尔什（Bill Walsh）和迪克·维梅尔（Dick Vermeil）。[3]斯巴达人的主教练是肯·尼乌马塔洛洛（Ken Niumatalolo）。圣何塞州立大学在CEFCU体育场（1933年建造，原名为斯巴达体育场）进行主场比赛，其区域对手包括弗雷斯诺州立大学、斯坦福大学、夏威夷大学和圣地亚哥州立大学。圣何塞州立大学于1892年组建了首支足球队，但没有教练。在组建美式足球队的第一个赛季，斯巴达人队唯一一场比赛便输给了当地圣何塞的YMCA队。比赛实际上是在1893年进行的，但它代表的是1892学年度，因此被计入1892年的赛季。[1]1983年，詹姆斯·E·阿迪科特（James E. Addicott）开始担任主教练。阿迪科特还在加利福尼亚州立师范学院（现为圣何塞州立大学）担任数学教授。该队在1893年和1894年与当地YMCA俱乐部进行了比赛，并在1896年获得了首场平局，以6比6战平了附近的对手太平洋学院。第一个常规赛赛季始于1898年，主要包括与当地高中和一些学院和社区学院的比赛。[4]在20世纪20年代，足球队开始在“斯巴达人球场”（Spartan Field）进行主场比赛，该球场后来成为斯巴达体育场（Spartan Stadium）的所在地。在20世纪30年代和40年代，斯巴达人足球队被认为是一支强大的球队，连续12个赛季取得胜利，并在18年的时间里获得了8个联盟冠军头衔。1932年和1939年的球队分别取得了7胜0负2平的战绩和13胜0负的战绩，这是学校历史上仅有的两次不败赛季。[4][3]圣何塞州立大学于1939年首次出现在全国排行榜上，当时美联社（AP）民意调查将斯巴达人队排在第七周的第19位。该队将在第八周升至第18位。劳埃德·托马斯（Lloyd Thomas）是首位获得全美一线队荣誉的圣何塞州立大学球员。托马斯在1936年、1937年和1938年的球队中担任防守端锋，这三支球队一共取得了27胜7负1平的战绩。截至2022年，圣何塞州立大学培养了90多名全美明星球员，包括7名一线队成员。[5]斯巴达体育场（现称为CEFCU体育场）建在原来的斯巴达人球场遗址上，于1933年完工，可容纳18000人。斯巴达人队在1933年10月7日，以44比6战胜了旧金山州立大学，赢得了该体育场的第一场足球比赛。圣何塞州立大学斯巴达人足球队在二战期间意外地加入了檀香山警察局。这支球队刚刚抵达夏威夷，准备参加一系列季后赛碗赛，对手是夏威夷大学彩虹勇士队和惠勒特大学熊猫队，结果1941年12月7日，珍珠港的美国海军基地遭到袭击。袭击事件发生后，这支球队被困在岛上好几个星期，球员们受雇于当地警察局，帮助加强岛屿防御，以防日本可能的登陆作战，以及担任岛上军事基地的警卫。[3][6]在20世纪50年代和60年代，斯巴达人足球队只取得了6个赛季的胜利，但20世纪70年代标志着一系列成功的赛季的开始，持续了20年。从1971年到1992年，
